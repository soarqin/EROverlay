# 更新日志

本项目所有重要变更均记录于此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

---

### [1.5.1] - 2025-11-13

#### 修复
- 未启用 `challenge_mode` 时，`死亡次数` 始终显示为 0 的问题。

### [1.5.0] - 2025-10-20

#### 变更
- Boss 覆盖层现在作为独立模块（`Boss.dll`）由覆盖层加载器加载。

### [1.4.0] - 2024-10-15

#### 新增
- 挑战模式：在 `[boss]` 配置节中将 `challenge_mode` 设置为 `true` 即可启用。
- 现在可以手动自定义显示文本格式，详见 `boss.ini`。

#### 变更
- 当字符集不是拉丁或西里尔文且未明确配置字体文件时，字体加载现在会回退到系统字体链接。

### [1.3.0] - 2024-07-03

#### 新增
- 在 `[common]` 配置节中添加 `language` 键，可独立于游戏语言选择数据语言。

#### 变更
- 更新数据文件夹结构，以支持多语言数据文件。
- `[boss]` 配置节中的 `data` 键重命名为 `data_file`（现在表示数据文件名）。
- Boss 内存地址现在通过 Flag ID 解析。

#### 移除
- 从 Boss 数据中移除 `boss_display` 和 `place_display` 字段（无实际用途）。

#### 修复
- Mod 卸载后鼠标光标仍被锁定的问题。
- 某些情况下玩家位置区域判断不正确的问题。
- 多处 Boss 标志数据修正。

### [1.2.0] - 2024-06-30

#### 新增
- 配置文件中新增 `[style]` 配置节，用于自定义覆盖层面板外观。
- 配置文件中新增 `[input]` 配置节，用于自定义快捷键。
- 未安装 DLC 时，DLC Boss 不再出现在列表中。
- 当 `[boss]` 中的 `data` 留空时，现在会根据游戏语言自动加载数据（注意：显示非拉丁字符时需手动设置 `[common]` 中的 `font` 和 `charset`）。

#### 变更
- 默认字体改为 **Open Sans**，默认字号设为 20，提升可读性。
- `[common]` 中的 `charset` 现在默认为空，会根据游戏语言自动选择字符集。

#### 修复
- 改进 DX12 钩子以减少崩溃并支持全屏模式（现在可以使用 Alt+Tab，但可能仍存在轻微渲染瑕疵）。
- 修复 Mod 可被多次同时加载的问题。

### [1.1.3] - 2024-06-24

#### 修复
- 由错误内存释放引起的崩溃。

#### 变更
- 切换至 [Pattern16](https://github.com/Dasaav-dsv/Pattern16) 以提升特征码扫描速度。

### [1.1.2] - 2024-06-23

#### 修复
- `劳弗古遗迹` 中的 `神兽舞狮` 未出现在 Boss 列表中。

### [1.1.1] - 2024-06-23

#### 修复
- 修改应用钩子补丁的时机判断逻辑，降低崩溃风险。
- 在加载画面或返回主菜单时不再检查 Boss 数据。

### [1.1.0] - 2024-06-22

#### 新增
- 支持带有 DLC「黄金树幽影」的艾尔登法环 v1.12，新增 41 个 Boss（已知全部 Boss，部分可选 Boss 可能遗漏）。

#### 修复
- 对原有 Boss 数据进行了若干修正。

### [1.0.1] - 2024-01-13

#### 新增
- 添加 `boss.panel_pos` 配置键，用于调整 Boss 面板位置。

#### 修复
- Boss 数据中的若干错误。

### [1.0.0] - 2023-08-27

#### 新增
- 首次发布。
